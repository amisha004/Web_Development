Internal Style Sheet Example

ID selectors: You can apply styles to a specific element with a unique ID using the pound (#) followed by the ID name. For example: #my-id { font-size: 20px; }

Class selectors: You can apply styles to elements with a specific class using the period (.) followed by the class name. For example: .my-class { color: red; }

Pseudo-class selectors: You can apply styles to elements based on their state, such as when they are being hovered over or clicked. For example: a:hover { text-decoration: underline; }
